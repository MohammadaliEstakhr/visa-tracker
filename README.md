# Visa Tracker

A simple and practical Flask web application to help users track their visa application submissions.

## ✨ Features

- Submit visa applications through a user-friendly HTML form
- Record personal information, visa type, application date, and current status
- Built with Flask and SQLite
- Clean structure for further feature expansion
